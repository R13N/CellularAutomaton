Cellular Automaton

Een cellulaire automaat (Engels: cellular automaton) is een discreet model uit de automatentheorie dat onder andere wordt toegepast in de wiskunde. Het model bestaat uit een één- of meer-dimensionaal raster van cellen met elk een eindig aantal toestanden. Een volgende toestand wordt door toepassing van een gegeven set regels berekend uit de huidige toestand van de cel en die van zijn directe buren.
